# jd-search-demo

使用 vue+vite 仿生京东搜索页

## 涉及技术

mock+axios+pina+vue-router4.0+NutUi

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

## 组件库

[NutUi](https://nutui.jd.com/#/zh-CN/guide/intro)
